Settings

activeClass

类型: string

默认: sturns_active

设置当前焦点页的样式类名

animate

类型: boolean

默认: true

轮播时，是否使用动画

autoplay

类型: boolean

默认: true

是否自动轮播

autoplaySpeed

类型: number

默认: 3000

轮播的间隔时间

boundary

类型: number

默认: 显示区域宽度的1/6

滑动页面时触发翻动的临界值

centerMode

类型: boolean

默认: false

是否初始化为居中模式

centerMode

类型: boolean

默认: false

是否初始化为居中模式

centerModeScale

类型: number

默认: 1

初始化为居中模式时候页面居中时候的缩放比

circle

类型: boolean

默认: true

是否初始化为循环模式

during

类型: number

默认: 400

页面轮播时的动画持续时间

centerMode

类型: boolean

默认: false

是否初始化为居中模式

direction

类型: string

默认: ltr

自动轮播时，ltr:从左向右（从上向下），rtl:从右向左（从下向上）

hoverStop

类型: boolean

默认: true

当鼠标悬停与轮播区域之上，是否停止自动轮播

isVertical

类型: boolean

默认: false

是否初始化为垂直轮播模式

pointer

类型: boolean

默认: true

是否自动生成页面指示器

pointerActiveClass

类型: string

默认: s_indi_active

当前页面指示器的样式类名

pointerClass

类型: string

默认: s_turnsPointerClass

页面指示器的默认样式类名

pointerPosition

类型: string

默认: s_indicator

页面指示器的位置样式类名

preventDefault

类型: boolean

默认: true

是否在滑动时，阻止默认事件

timingFn

类型: string

默认: ease-out

轮播时，动画应用的动画时间函数，有ease，ease-in，ease-out，ease-in-out，还可以自行指定贝塞尔曲线函数，如：cubic-bezier(n,n,n,n)

direction

类型: string

默认: ltr

自动轮播时，ltr:从左向右（从上向下），rtl:从右向左（从下向上）

direction

类型: string

默认: ltr

自动轮播时，ltr:从左向右（从上向下），rtl:从右向左（从下向上）

Propertys

currentIndex

类型: number

当前页面序号

mainEleW

类型: number

轮播区域的宽度

mainEleW

类型: number

轮播区域的高度

Events

onScroll

类型: function

默认: undefined

轮播时的回调函数

onScrollBeforeStart

类型: function

默认: undefined

轮播前的回调函数

onScrollEnd

类型: function

默认: undefined

轮播完成后的回调函数

onScrollStart

类型: function

默认: undefined

轮播开始时的回调函数

Function

prev

轮播到上一页

next

轮播到下一页

scrollBy

参数: number

相对滚动多少像素

scrollTo

参数: number

相对于轮播父元素滚动多少像素

scrollToPage

参数: number

滚动到第几页

start

参数: 无

开始轮播

stop

参数: 无

停止轮播
